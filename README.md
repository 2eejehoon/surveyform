# [클라썸] 구글 설문조사 Clone 만들기

## 실행 방법

```
npm install
npm run start
```

## 스킬

- React
- Redux-Toolkit
- TypeScript
- Styled-Components

## 구현 기능

- 설문지 제목 추가, 편집 (O)
- 설문지 설명 추가, 편집 (O)
- 질문을 추가하면 질문이 추가됩니다. (O)
  - 단답형 (O)
  - 장문형 (O)
  - 객관식 질문 (O)
  - 체크박스 (O)
  - 드롭다운 (O)
- 질문 복사 기능 (O)
- 질문 삭제 기능 (O)
- 필수 옵션 설정 기능 (O)
- 미리 보기 기능 (O)
- 제출 기능 (O)
- 양식 지우기 기능 (O)
- 질문 순서 변경 (X)
- 질문의 옵션 순서 변경 (X)
